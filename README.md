# PairBoost

Implementation and experiments on the PairBoost algorithm from the "Classification from Pairwise Comparisons" draft

Algorithm implementation code in `main.py`. Correct experiments on datasets from "Classification from Pairwise Similarity and Unlabeled Data" and "Classification from Pairwise Similarities/Dissimilarities and Unlabeled Data via Empirical Risk Minization" in `SVM_ranked_data_experiments` directory. 

Data in google drive folder [here](https://drive.google.com/open?id=1IXOYO9TMI9jnQirYYiLuaxNbt3BTYipw), full results in google sheet [here](https://docs.google.com/spreadsheets/d/1i_ab6YPXRmFcW7HaL8XPvjHhS8W7q5kRWYXfIJi8KYk/edit?usp=sharing)
